# 날짜별 체크리스트 윈도우 위젯

## 설치 방법 (WinWidgets 사용)

### 1단계 — WinWidgets 설치
https://github.com/beyluta/WinWidgets/releases
에서 최신 버전 다운로드 후 설치

### 2단계 — 위젯 폴더 복사
이 폴더(checklist_widget) 전체를 아래 경로에 복사:
  C:\Users\[사용자이름]\AppData\Local\WinWidgets\widgets\

### 3단계 — 위젯 실행
WinWidgets 앱 실행 → 위젯 목록에서 "날짜별 체크리스트" 선택 → 활성화

---

## 대안: 브라우저를 위젯처럼 사용

checklist.html 파일을 Chrome/Edge에서 열고
창 크기를 조절한 뒤 항상 위에 표시(Always on top) 설정하면
위젯처럼 사용 가능합니다.

Chrome 확장 프로그램 "Always on Top" 설치 후 사용 권장.

---

## 기능
- 데일리 루틴 1 / 2 (자정마다 자동 초기화)
- 오늘 / 내일 / 모레 3일치 체크리스트
- 미완료 항목 자동 이월 (다음날로 넘김)
- ◁ ▷ 버튼으로 날짜 이동
- localStorage에 데이터 자동 저장
